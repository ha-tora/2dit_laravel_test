### Быстрый старт:

1. **Для первого запуска прописать команду**:
```bash
make start
```

2. **Сборка контейнеров**:
```bash
make build
```

3. **Запуск контейнеров**:
```bash
make up
```

4. **Остановка контейнеров**:
```bash
make down
```

5. **Запуск миграций**:
```bash
make migrate
```

6. **Создать админа**:
```bash
make admin
```
